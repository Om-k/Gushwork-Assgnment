Mangalam HDPE Pipes - Landing Page
A responsive, high-performance landing page built with Vanilla HTML, CSS, and JavaScript.

✨ Key Features
Sticky Header: Slides into view after the first fold; hides on scroll-down and reveals on scroll-up.
App Carousel: Interactive slider with a custom start offset and responsive card counts.
Zoom Preview: Real-time hover preview for carousel images that follows the cursor.
Smooth FAQ: Accordion using CSS Grid transitions to prevent layout shifts.
Pixel Perfect: Matches Figma design specifications for desktop, tablet, and mobile.

📁 Project Structure
index.html - Semantic markup and metadata.
styles.css - Custom properties, Flexbox/Grid, and animations.
script.js - Carousel logic, scroll handlers, and UI toggles.
/Assets - Optimized SVGs, favicons, and product images.

🚀 How to Run
Download or clone the project.
Open index.html in any browser.
Recommended: Use VS Code Live Server to ensure all SVG paths load correctly.
